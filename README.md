# Sistema de Contra Cheques — IECPN

Sistema web responsivo para gerenciamento e consulta de contracheques desenvolvido para o Instituto Estadual do Cérebro Paulo Niemeyer (IECPN).

---

# Sobre o Projeto

O sistema permite:

- Login de colaboradores
- Login administrativo
- Cadastro de usuários
- Upload de contracheques em PDF
- Download de holerites
- Confirmação de ciência dos documentos
- Histórico de contracheques
- Painel administrativo com gestão de colaboradores

Todo o sistema funciona diretamente no navegador utilizando:

- HTML5
- CSS3
- JavaScript Vanilla
- LocalStorage

# Funcionalidades

# Área do Colaborador

- Login via CPF
- Visualização de contracheques
- Download de PDFs
- Confirmação de ciência
- Histórico completo
- Busca por mês ou arquivo

# Área Administrativa

- Login administrador
- Cadastro automático de colaboradores
- Upload de PDFs
- Controle de ciência dos usuários
- Consulta detalhada dos colaboradores
- Exclusão e edição de usuários
